# Homebrew Tap for sqry

Semantic code search tool — [sqry.dev](https://sqry.dev)

## Install

```bash
brew install verivus-oss/sqry/sqry
```

Or tap first:

```bash
brew tap verivus-oss/sqry
brew install sqry
```

## Update

```bash
brew upgrade sqry
```
